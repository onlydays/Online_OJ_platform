# 前端优化方案 v2.0

> 基于现有后端 API，不修改任何后端代码

---

## 1. 登录后显示用户名

**现状**：导航栏只显示"登录/退出"链接，登录后不显示用户名。

**改造**：

```
当前导航栏:  [Online OJ]  题目  记录  个人  [登录]
改造后:      [Online OJ]  题目  记录  个人  管理   [👤 admin] [退出]
```

| 状态 | 显示 |
|------|------|
| 未登录 | `[登录]` 按钮 |
| 已登录 | 用户名 + 头像圆圈(首字母) + `[退出]` |
| 管理员 | 额外显示 `[管理]` 链接 |

**实现位置**：`auth.js` 的 `updateNav()` 方法。

## 2. LeetCode 风格界面优化

参考 LeetCode 的视觉设计，保持现有后端 API 不变：

### 2.1 配色方案
```
主背景:   #fafafa (LeetCode 暖灰)
卡片:     #ffffff
主色调:   #2cbb5d (LeetCode 绿)
次要色:   #007aff
文字色:   #262626
边框色:   #e5e5e5
难度-简单: #00b8a3
难度-中等: #ffc01e
难度-困难: #ff375f
```

### 2.2 布局调整
- 导航栏改为白色 + 底部阴影
- 题目列表/代码编辑区改为 LeetCode 式卡片
- 题目详情页改为 Markdown 风格
- 按钮改为圆角胶囊型

### 2.3 组件改造清单

| 组件 | 改造点 |
|------|--------|
| 导航栏 | 白色背景 + 底阴影 + 用户头像 |
| 题目列表 | 难度标签改为 LeetCode 配色 |
| 题目详情 | 描述区加 Markdown 样式 |
| 提交结果 | 状态徽章改为 LeetCode 风格 |
| 登录页 | 居中卡片 + 品牌色按钮 |
| 个人主页 | 统计卡片改为圆角大数字 |

## 3. 退出登录 → 跳转登录页

**现状**：`auth.js` 的 `logout()` 已实现跳转 `#/login`。

**验证**：`localStorage.clear()` + `window.location.hash = '#/login'` ✓

额外增强：未登录访问需要登录的页面时，自动跳转 `#/login`。
